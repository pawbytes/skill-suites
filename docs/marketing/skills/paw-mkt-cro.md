# `marketing-cro`

## Overview

Improves conversion across landing pages, flows, onboarding, paywalls, and forms.

## When to use it

Use `/paw-mkt-cro` when you need:
- landing page improvements
- signup or checkout optimization
- onboarding activation analysis
- test ideas backed by funnel evidence

## Inputs to prepare

- current page or flow
- conversion goals
- funnel metrics
- user objections or friction points
- screenshots or page links if available

## Deliverables

- CRO hypotheses
- page or flow recommendations
- experiment plans
- optimization audits

## Output locations

```text
brands/{brand-slug}/campaigns/cro/
```

## Related skills

- `marketing-analytics`
- `marketing-paid-ads`
- `marketing-seo`
- `marketing-sales`
- `marketing-psychology`

## Sample prompts

```text
/paw-mkt-cro
Audit our homepage-to-signup flow.
```

```text
/paw-mkt-cro
We get traffic but weak trial conversion. Build a prioritized CRO plan for our pricing and signup pages.
```

```text
/paw-mkt-cro
Use our audience objections and product marketing context to improve our webinar landing page.
```
