### The Coordinator

All marketing requests flow through the **Marketing Agency Coordinator**, which:
- Routes your request to the right specialist
- Manages brand workspaces
- Tracks progress across campaigns
- Orchestrates multiple specialists when needed

## Key Concepts

### SOSTAC Framework

The strategic backbone of the system:
- **S**ituation - Where are we now?
- **O**bjectives - Where do we want to be?
- **S**trategy - How do we get there?
- **T**actics - What tools do we use?
- **A**ction - How do we execute?
- **C**ontrol - How do we measure?

### Brand Workspace

Every brand gets a dedicated workspace where all plans, content, and deliverables are saved:
```
.pawbytes/marketing-suites/brands/{your-brand}/
├── brand-context.md
├── paw-mkt-product-context.md
├── sostac/
├── campaigns/
├── channels/
└── operations/
```

### Progressive Disclosure

Skills load information on-demand rather than all at once—saving tokens and providing relevant information when you need it.
