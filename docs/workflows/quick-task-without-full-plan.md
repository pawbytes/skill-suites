# Workflow: Quick Task Without a Full Plan

Use this workflow when you need one deliverable now and do not want to stop for a full planning cycle.

## Best starting commands

```text
/paw-mkt-agency
```

or a direct specialist such as:

```text
/paw-mkt-social
/paw-mkt-email
/paw-mkt-seo
/paw-mkt-cro
```

## When this path makes sense

- you need one asset quickly
- the brand has enough context already
- you are exploring before committing to a full strategy
- the work is narrow and time-sensitive

## Trade-off

This path is faster, but the suite may still recommend SOSTAC if it detects that strategy is missing.

## Typical examples

- one LinkedIn post
- one landing page rewrite
- one email sequence
- one keyword brief
- one Product Hunt launch checklist

## Suggested minimum context to provide

Even for a quick task, try to include:
- brand name
- audience
- offer or feature
- desired outcome
- channel
- constraints or deadline

## Sample prompts

### Basic
```text
/paw-mkt-social
Write one LinkedIn post for our new feature launch.
```

### With brand context
```text
/paw-mkt-email
For Northstar AI, write a 4-email nurture sequence for talent leaders who downloaded our hiring scorecard.
```

### Coordinator route
```text
/paw-mkt-agency
I don't want a full strategy right now. I just need a conversion-focused landing page outline for our webinar signup.
```

## What usually happens next

If the quick task works and you want to scale:
- create or refresh `product-marketing-context.md` (via `/paw-mkt-product-context`)
- run `/paw-mkt-sostac`
- move into broader implementation
