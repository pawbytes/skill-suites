# `marketing-pr`

## Overview

Supports earned media, press outreach, narrative shaping, and reputation-sensitive communications.

## When to use it

Use `/paw-mkt-pr` when you need:
- a press release
- media outreach strategy
- a media list
- thought leadership pitching
- brand reputation support

## Inputs to prepare

- announcement or angle
- spokesperson or source material
- proof points
- target publications or audiences

## Deliverables

- press releases
- cold pitch emails and pitch types guide
- media lists
- HARO and Connectively responses
- follow-up sequences
- podcast pitches
- speaking engagement pitches
- media kits
- crisis statements
- spokesperson preparation guides
- performance summaries

## Framework files

12 individual framework files, indexed in `frameworks-index.csv`:

- Cold pitch template
- Pitch types guide (news, data, opinion, etc.)
- Pitch checklist
- HARO/Connectively response template
- Follow-up sequence
- Podcast pitch template
- Speaking pitch template
- Press release template
- Media kit checklist
- Crisis statement templates
- Spokesperson preparation

## Output locations

```text
brands/{brand-slug}/campaigns/pr/
```

Often using subfolders such as:
- `press-releases/`
- `media-lists/`
- `performance/`

## Related skills

- `marketing-launch`
- `marketing-influencer`
- `marketing-social`
