# How to Use These Frameworks

1. **Select the framework** that matches your business model and goal
2. **Customize timing** based on your sales cycle and audience data
3. **Adapt subject lines** as starting points, then A/B test variations
4. **Implement branching gradually** -- start with core sequence, add branches as data comes in
5. **Measure against benchmarks** in the companion file (benchmarks.md)
6. **Iterate quarterly** -- review performance, update copy, test new structures

> Every email in a sequence should be valuable on its own. Don't create dependencies
> where missing one email makes the next one confusing.
