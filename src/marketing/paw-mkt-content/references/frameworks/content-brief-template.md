# Content Brief Template

### Required Fields

```
CONTENT BRIEF
=============

1. TARGET KEYWORD
   Primary: [main keyword, search volume, keyword difficulty]
   Secondary: [2-5 related keywords]
   LSI/Related terms: [8-12 semantically related terms to include naturally]

2. SEARCH INTENT
   Type: [Informational / Commercial / Transactional / Navigational]
   What the searcher wants: [1-2 sentence description]
   What they should do after reading: [desired next action]

3. CONTENT SPECIFICATIONS
   Format: [Blog post / Guide / Listicle / Comparison / How-to / etc.]
   Target word count: [range, e.g., 1,800-2,200 words]
   Funnel stage: [TOFU / MOFU / BOFU]
   Content pillar: [which topic cluster this belongs to]

4. OUTLINE
   H1: [Title -- include primary keyword]
   H2: [Section 1]
     H3: [Subsection]
     H3: [Subsection]
   H2: [Section 2]
     H3: [Subsection]
   [... continue for all sections]
   H2: [Conclusion / Next Steps / CTA section]

5. COMPETITIVE ANALYSIS
   Top 3 ranking pages:
     - [URL 1]: [word count, strengths, weaknesses, gaps]
     - [URL 2]: [word count, strengths, weaknesses, gaps]
     - [URL 3]: [word count, strengths, weaknesses, gaps]
   Our differentiation angle: [how we will be better/different]

6. INTERNAL LINKS
   Link to: [3-5 existing pages on our site to link to]
   Link from: [existing pages that should link to this new piece]
   Pillar page: [URL of the parent pillar page if applicable]

7. CTA (Call to Action)
   Primary CTA: [what we want the reader to do]
   Secondary CTA: [newsletter signup, related content, etc.]
   Lead magnet: [downloadable asset if applicable]

8. STYLE AND VOICE
   Tone: [Educational / Conversational / Authoritative / etc.]
   Audience: [who is reading this, their expertise level]
   Dos: [specific guidance, e.g., "use real examples", "include data"]
   Don'ts: [what to avoid, e.g., "no jargon without explanation"]

9. ASSETS NEEDED
   [ ] Custom graphics/illustrations
   [ ] Screenshots or product images
   [ ] Data charts or infographics
   [ ] Video embed
   [ ] Downloadable template/resource
   [ ] Author headshot and bio

10. SEO CHECKLIST
    [ ] Primary keyword in H1
    [ ] Primary keyword in first 100 words
    [ ] Primary keyword in meta title (under 60 chars)
    [ ] Meta description written (under 155 chars, includes keyword)
    [ ] Secondary keywords used naturally in H2s
    [ ] Image alt text includes relevant keywords
    [ ] URL slug is short and includes keyword
```
