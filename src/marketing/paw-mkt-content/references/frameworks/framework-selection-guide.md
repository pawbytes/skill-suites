# Quick Reference: Choosing the Right Framework

| Situation | Framework to Use |
|-----------|-----------------|
| Building out SEO content strategy | Topic Cluster Model |
| Deciding what content to create for a campaign | Content Funnel Mapping |
| Competing for a high-value keyword | Skyscraper Technique 2.0 |
| Maximizing ROI of a single content piece | Content Repurposing Framework |
| Evaluating whether to publish a piece | Content Scoring Model |
| Planning monthly and quarterly output | Editorial Calendar Framework |
| Writing a customer success story | Case Study Framework (STAR) |
| Assigning a content piece to a writer | Content Brief Template |
| Incorporating AI into your workflow | AI Content Workflow |
| Writing conversion copy for landing/pricing pages | Conversion Copywriting Framework |
