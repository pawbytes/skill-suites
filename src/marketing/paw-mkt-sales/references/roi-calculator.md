# ROI Calculator Capability

**Outcome:** Build ROI models using prospect's own numbers for maximum credibility, delivered in Google Sheet, embedded calculator, or whiteboard format.

---

## When to Use

Proposal support, CFO/finance approval conversations, business case justification, board presentations.

---

## Core Formula Chain

```
Time saved/year = (hours/week x team members x 52) x efficiency gain %
Labor savings = time saved x fully-loaded hourly rate
Total annual value = labor savings + tool cost delta + error cost eliminated
Net ROI = (total annual value - investment) / investment x 100
Payback period = investment / (total annual value / 12) months
```

---

## Delivery Formats

| Format | When to Use | Specs |
|---|---|---|
| Google Sheet | In-meeting, editable with prospect numbers | Light-blue input cells, formula outputs, plain-language summary |
| Embedded calculator | Pricing page, self-service | 3-5 sliders (team size, hours, rate) with instant output + demo CTA |
| In-meeting whiteboard | Live call, no prep time | 3 questions: team size, hours/week, hourly cost |

---

## Common Mistakes to Avoid

- Never substitute your assumptions for their numbers
- Round to nearest $1,000 — false precision reduces credibility
- Pick 2-3 value drivers, not every possible benefit
- Always frame result as "conservative"
