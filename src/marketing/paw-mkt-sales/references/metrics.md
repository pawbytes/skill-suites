# Metrics Tracking Capability

**Outcome:** Track which collateral drives deals, identify what's working, and run quarterly audits to retire unused assets.

---

## When to Use

Setting up enablement measurement, monthly performance review, quarterly sales team audit.

---

## Key Metrics

| Metric | What It Measures | How to Track |
|---|---|---|
| Collateral usage rate | Assets reps share vs what exists | CRM attachment, Highspot/Seismic analytics |
| Content-to-close correlation | Assets in won vs lost deals | Asset logging in CRM deal notes |
| Time to find asset | If >2 min, discoverability problem | Shadow reps, ask directly |
| Deal stage conversion by asset | Does ROI calculator improve Proposal→Close? | CRM stage with asset tag |
| Objection frequency by stage | Which objections, when | CRM post-meeting notes |
| Champion kit adoption | How often champions use kit vs fly solo | Post-deal review |

---

## Quarterly Audit Questions

Ask sales team every quarter:

- Which 3 assets did you use most?
- Which asset would you never use? Why?
- What objection did you struggle to handle? Did you have a resource?
- What competitor came up most? Do you feel equipped?

---

## Action from Audit

Use answers to:
1. Retire what's not working
2. Build what's missing
3. Improve what's used but underperforming
