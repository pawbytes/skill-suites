## Program Setup

### 1. Choose a Platform

| Platform | Best For | Key Features |
|---|---|---|
| Shopify Collabs | Shopify stores | Direct integration, product seeding |
| TikTok Shop Affiliate | TikTok commerce | In-app checkout, algorithm-driven discovery |
| Amazon Associates | Amazon sellers | Massive product catalog, trusted platform |
| ShareASale | Multi-brand | Established network, diverse merchants |
| Impact | Performance focus | Advanced tracking, partnership management |
| Refersion | Shopify/WooCommerce | Ecommerce-focused, easy setup |
| Custom tracking | Unique needs | Full control, custom attribution |

### 2. Define Commission

| Model | Structure | Best For |
|---|---|---|
| Flat percentage | 10-30% of sale | Simple, scalable, most common |
| Tiered percentage | 10% base, 15% at $5K/mo, 20% at $10K/mo | Incentivizes volume |
| Flat fee per conversion | $5-$50 per sale or lead | SaaS, subscription, lead gen |
| Hybrid | Base stipend + commission | Consistent posting motivation |
| Time-limited boost | 25% first month, 15% ongoing | Launch promotions |

### 3. Create Tracking Assets

- Unique discount codes per creator (CREATOR15, JESS20)
- UTM-tagged affiliate links: `?utm_source=influencer&utm_medium={platform}&utm_campaign={campaign}&utm_content={creator}`
- Platform-native tracking (TikTok Shop, Shopify Collabs)

### 4. Set Attribution Window

- 7-30 days standard
- Longer windows favor creators, shorter reduce over-attribution
- 14-30 days recommended for most products

### 5. Build Creator Portal

Components:
- Application form
- Onboarding materials
- Asset library (logos, images, guidelines)
- Performance dashboard
- Payment schedule

## TikTok Shop Affiliates

### Setup Process

1. Register as TikTok Shop seller
2. Add products to affiliate marketplace
3. Set commission rates (15-25% for competitive recruitment)
4. Creators browse and select products for shoppable videos and Lives
5. In-app checkout reduces friction

### Key Statistics

- TikTok Shop affiliate engagement rates: 5.2% (160% higher than Instagram)
- Creators with under 50K followers achieve 30.1% engagement on affiliate content
- Conversion rates: 2-8% in-app (significantly higher than website redirect)
- Algorithm surfaces shoppable content to high-intent users

### Recruitment Strategy

- Set competitive commission (15-25%)
- Optimize product listings for TikTok search
- Direct outreach to creators with elevated commission offers
- Leverage TikTok Live for real-time selling
