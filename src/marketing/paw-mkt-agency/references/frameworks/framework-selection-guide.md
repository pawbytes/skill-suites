# Framework Selection Guide

Use this decision tree to select the right framework for the task at hand.

| If the client needs... | Use this framework |
|---|---|
| Full lifecycle marketing plan | RACE (integrated with SOSTAC) |
| Audience-intent content strategy | See-Think-Do-Care |
| Channel discovery and prioritization | Bullseye Framework |
| Experiment prioritization | ICE Scoring |
| Product-led growth metrics | AARRR Pirate Metrics |
| Self-reinforcing growth strategy | Growth Loops |
| Understanding customer experience | Customer Journey Mapping |
| Assessing marketing sophistication | Marketing Maturity Model |
| Budget planning and allocation | 70/20/10 + Business Stage Model |
| Overall strategic marketing plan | SOSTAC (primary skill framework) |

## Combining Frameworks (Recommended Stack)
For a comprehensive agency engagement, layer frameworks as follows:

1. **Marketing Maturity Assessment** — Understand where the client is today.
2. **SOSTAC** — Build the strategic plan (Situation, Objectives, Strategy, Tactics, Action, Control).
3. **RACE** — Structure execution across the customer lifecycle.
4. **See-Think-Do-Care** — Map content and messaging to audience intent.
5. **Bullseye** — Prioritize channels through rapid testing.
6. **ICE** — Prioritize experiments and optimizations within each channel.
7. **AARRR** — Track metrics across the full customer lifecycle.
8. **70/20/10** — Allocate budget with structured risk management.
9. **Customer Journey Mapping** — Continuously improve the experience.
10. **Growth Loops** — Identify and engineer compounding growth mechanisms.
