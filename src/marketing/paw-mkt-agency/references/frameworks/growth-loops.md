# Growth Loops vs. Funnels

## The Funnel Model
- Linear: Awareness -> Interest -> Desire -> Action (AIDA) or TOFU -> MOFU -> BOFU.
- Each stage loses a percentage of prospects (leaky bucket).
- Requires constant top-of-funnel investment to sustain growth.
- Best for: Product launches, new market entry, campaign planning, sales-led organizations.

## Growth Loops (Flywheel / Compounding Growth)
- Non-linear: Output of one user action becomes the input for acquiring/activating new users.
- Self-reinforcing: Growth compounds rather than leaking.
- Reduces dependency on paid acquisition over time.

## Types of Growth Loops

**Viral Loop**:
- User creates content or invites others -> New users sign up -> They create/invite -> Repeat.
- Example: Dropbox referral program (give storage, get storage).
- Key metric: Viral coefficient (K-factor). K > 1 = exponential growth.

**Content Loop**:
- Users create content -> Content indexed by search engines -> New users discover content -> They create content -> Repeat.
- Example: Pinterest (users pin content -> SEO traffic -> new users -> more pins).
- Key metric: Organic traffic growth rate, UGC volume.

**Paid Loop**:
- Revenue from customers -> Reinvested into ads -> Ads acquire new customers -> Revenue -> Repeat.
- Example: Ecommerce brand reinvesting 20% of revenue into Meta Ads.
- Key metric: LTV:CAC ratio (target: 3:1 or higher for sustainable loop).

**Sales Loop**:
- Happy customers -> Referrals / case studies -> Credibility attracts new customers -> Repeat.
- Example: B2B SaaS using customer success stories to close new deals.
- Key metric: Referral rate, case study conversion impact.

## When to Use Each
| Scenario | Use Funnel | Use Growth Loop |
|---|---|---|
| New product, no traction | Yes | Not yet |
| Established product with network effects | Support | Primary |
| Sales-led B2B with long cycles | Primary | Support |
| Product-led growth (PLG) model | Support | Primary |
| Ecommerce with repeat purchase | Hybrid | Paid loop |
| Content/community platform | Support | Content loop |

## How to Identify Growth Loops in a Business
1. Map the customer journey from acquisition to activation to monetization.
2. Identify where output from one stage could feed back into an earlier stage.
3. Look for: user-generated content, referral incentives, viral mechanics, data network effects.
4. Ask: "Does each new customer make the product more valuable for the next customer?"
5. If no natural loop exists, engineer one (referral program, affiliate program, content program).
