# ICE Scoring (Sean Ellis)

**What it is:** A prioritisation framework scoring tactical ideas on Impact, Confidence, and Ease to produce a ranked backlog.

**SOSTAC Phase:** Tactics

**Formula:** `ICE Score = (Impact + Confidence + Ease) / 3`

**Scoring guide (1-10 each):**
- **Impact:** If this works, how much will it move the key metric?
- **Confidence:** How certain are you it will work (based on data, tests, analogues)?
- **Ease:** How quick and cheap is it to implement?

**ICE-scored tactic table template:**
| Tactic | Impact | Confidence | Ease | ICE Score | Owner | Status |
|---|---|---|---|---|---|---|
| | | | | | | |

**How to use:** Score a backlog of 10-20 candidate tactics. Work top-down from highest ICE score. Re-score after each sprint based on new learnings.
