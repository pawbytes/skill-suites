# Porter's Five Forces

**What it is:** Assesses the structural attractiveness of an industry by rating five competitive forces.

**SOSTAC Phase:** Situation

**Forces + Digital Business Assessment:**
| Force | Key Digital Questions | Rate: Low/Med/High |
|---|---|---|
| Threat of New Entrants | Low barriers to build online? VC-funded challengers entering? | |
| Supplier Power | Platform dependency (Meta, Google, Amazon)? API lock-in? | |
| Buyer Power | Low switching cost? Price comparison easy? Bulk buyers? | |
| Threat of Substitutes | Can customers solve the job with a different category entirely? | |
| Competitive Rivalry | Number of direct rivals? Price wars? Differentiation low? | |

**Output:** Force intensity rating per row + one-sentence strategic implication. High rivalry + high substitutes = must differentiate hard. High supplier power = diversify acquisition channels urgently.
