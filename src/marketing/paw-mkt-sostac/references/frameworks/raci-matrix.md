# RACI Matrix

**What it is:** A responsibility assignment matrix — Responsible (does the work), Accountable (owns the outcome), Consulted (input required), Informed (kept updated).

**SOSTAC Phase:** Action

**RACI rules:**
- Every task must have exactly one Accountable. Multiple A's = no clear ownership.
- Not everyone needs to be Consulted — over-consulting creates bottlenecks.
- Informed roles receive outputs, not drafts.

**Template — standard campaign launch:**
| Task | Marketing Lead | Designer | Developer | Exec Sponsor | Analytics |
|---|---|---|---|---|---|
| Campaign brief | A/R | C | C | C | I |
| Creative production | C | R | I | I | I |
| Landing page build | C | C | R | I | C |
| Campaign launch | A | I | R | I | R |
| Performance report | A/R | I | I | I | C |
