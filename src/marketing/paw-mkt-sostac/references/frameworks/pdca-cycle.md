# PDCA Cycle (Plan-Do-Check-Act)

**What it is:** A four-stage iterative improvement loop — Plan (set hypothesis and goal), Do (execute), Check (measure results), Act (standardise if successful, or revise and repeat).

**SOSTAC Phase:** Control

**Marketing cadences:**
- **Weekly PDCA:** Paid campaign optimisation (bid adjustments, ad copy, audience)
- **Monthly PDCA:** Content performance review, email programme, SEO progress
- **Quarterly PDCA:** Full channel mix review, OKR scoring, strategy adjustment

**PDCA log template:**
| Date | Plan (hypothesis) | Do (what ran) | Check (result vs. goal) | Act (decision) |
|---|---|---|---|---|
| | | | | |

**Usage note:** PDCA is the operating rhythm underneath the SOSTAC plan. Implement as a recurring team ritual, not a one-off exercise.
