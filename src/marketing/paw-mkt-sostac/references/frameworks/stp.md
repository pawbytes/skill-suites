# STP: Segmentation, Targeting, Positioning

**What it is:** The core strategic sequence — divide the market into segments, choose which to pursue, then define how to position against competitors in the chosen segment.

**SOSTAC Phase:** Strategy

**Segmentation variables:**
| Type | Variables |
|---|---|
| Demographic | Age, income, gender, family stage, job title |
| Psychographic | Values, lifestyle, personality, attitudes |
| Behavioural | Usage rate, loyalty, occasion, benefit sought |
| Technographic | Device, platform, tech stack (B2B) |
| Firmographic | Company size, industry, revenue, growth stage (B2B) |
| Needs-based | Primary job, pain severity, desired outcome |

**Segment attractiveness scoring matrix:**
| Criterion | Weight | Score 1-5 | Weighted |
|---|---|---|---|
| Market size | 20% | | |
| Growth rate | 20% | | |
| Competitive intensity | 15% | | |
| Strategic fit | 20% | | |
| LTV potential | 15% | | |
| Cost to reach | 10% | | |

**Targeting models:**
- **Undifferentiated:** One offer for all (commodities, mass FMCG)
- **Differentiated:** Separate offers per segment (requires resource)
- **Concentrated/Niche:** All resources on one segment (best for SMBs and startups)
- **Micromarketing:** Hyper-local or individual personalisation (requires data maturity)

**Positioning — perceptual map:** Choose two axes that matter most to target customers (not to the brand). Plot competitors and identify the open, desirable space. Axes should come from customer research (JTBD interviews, surveys), not gut feel.
