# AARRR Pirate Metrics (Dave McClure)

**What it is:** A five-stage funnel model — Acquisition, Activation, Retention, Revenue, Referral — used to diagnose where the biggest growth lever sits.

**SOSTAC Phase:** Tactics

**Diagnosing the biggest leak:** Calculate conversion rate between each stage. The stage with the worst conversion relative to benchmark is the priority for tactical investment.

| Stage | Definition | Benchmark range |
|---|---|---|
| **Acquisition** | Visitors from any channel | Cost per visitor varies widely |
| **Activation** | First "aha" moment / meaningful action | 20–40% of new visitors |
| **Retention** | Return visits / repeat usage | Varies by model; <20% is poor |
| **Revenue** | Paying conversion | 1–5% visitor-to-pay (direct) |
| **Referral** | Advocates who bring others | NPS 40+ usually correlates |

**AARRR → tactical priority:** Fix Activation before spending more on Acquisition. Fix Retention before scaling Revenue. A leaky bucket wastes acquisition spend.
