# Digital Maturity Model (5S)

**What it is:** Baseline assessment of how well the business currently uses digital marketing across five dimensions (PR Smith's 5Ss).

**SOSTAC Phase:** Situation

**Scoring guide (1 = none/ad hoc, 5 = systematic/optimised):**
| Dimension | What to Assess | Score 1-5 |
|---|---|---|
| **Sell** | Online revenue, conversion rate, e-commerce capability | |
| **Serve** | Digital customer service, self-serve, response time | |
| **Speak** | Content, social, email, SEO, two-way dialogue | |
| **Save** | Automation, efficiency gains from digital tools | |
| **Sizzle** | Brand experience, differentiation online, delight moments | |

**Output:** Radar chart (or table) of current vs. target maturity. Gaps become Situation findings feeding into Objectives.
