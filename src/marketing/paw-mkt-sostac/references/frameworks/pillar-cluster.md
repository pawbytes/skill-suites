# Pillar-Cluster SEO Model

**What it is:** A site architecture and content strategy where a broad "pillar page" covers a topic comprehensively and supporting "cluster pages" cover sub-topics, all linking back to the pillar.

**SOSTAC Phase:** Tactics

**Architecture:**
```
Pillar Page: "Complete Guide to [Core Topic]" (2,000–5,000 words)
  └── Cluster: "[Sub-topic 1]" — links to pillar
  └── Cluster: "[Sub-topic 2]" — links to pillar
  └── Cluster: "[Sub-topic 3]" — links to pillar
  └── Cluster: "[Sub-topic N]" — links to pillar
```

**Planning steps:** (1) Identify 3-5 core topic pillars aligned to buyer journey stages. (2) Keyword research each pillar for 10-20 cluster topic ideas. (3) Audit existing content — map to pillars or flag for deletion/consolidation. (4) Build internal linking plan.
