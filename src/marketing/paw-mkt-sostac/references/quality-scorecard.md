# SOSTAC Quality Scorecard

**Purpose:** Final plan review session to ensure industry-standard quality before execution.

## When to Use

After all 6 SOSTAC phases are complete, run this structured review session before marking the plan as final.

---

## Phase Quality Criteria

### Phase 1: Situation Analysis (Score X/10)

**Must Have:**
- [ ] Clear market sizing (TAM/SAM/SOM with sources)
- [ ] Competitor analysis with specific competitors named
- [ ] SWOT analysis with actionable insights
- [ ] TOWS matrix linking threats/opportunities to strategies
- [ ] 5S digital baseline (Sell, Speak, Save, Sizzle, Serve)
- [ ] Evidence-backed findings (not assumptions)

**Red Flags:**
- Vague market size ("large market")
- Generic competitors ("big companies")
- SWOT without strategic implications
- No data sources cited
- Assumptions stated as facts

---

### Phase 2: Objectives (Score X/10)

**Must Have:**
- [ ] North Star Metric defined
- [ ] 3-5 OKRs with measurable Key Results
- [ ] Targets with timeframes (Q2, Q4, etc.)
- [ ] RACE funnel alignment (Reach, Act, Convert, Engage)
- [ ] Realistic based on current capacity/budget
- [ ] Benchmarks or industry comparisons

**Red Flags:**
- Vague objectives ("grow revenue")
- No timeframes
- Impossible targets given constraints
- Missing funnel stage metrics
- No baseline to measure from

---

### Phase 3: Strategy (Score X/10)

**Must Have:**
- [ ] Clear positioning statement (Moore's formula)
- [ ] Defined target segments (STP)
- [ ] Value proposition for each segment
- [ ] Strategic pillars (3-5 max)
- [ ] Competitive differentiation articulated
- [ ] Customer journey mapped

**Red Flags:**
- "Everyone" is the target
- No clear differentiation
- Strategy doesn't address Situation gaps
- Tactics disguised as strategy
- Missing customer perspective

---

### Phase 4: Tactics (Score X/10)

**Must Have:**
- [ ] Prioritized channel mix with rationale
- [ ] ICE scores for major initiatives
- [ ] Content pillars/topics defined
- [ ] Budget allocation by channel
- [ ] Funnel stage mapping (TOFU/MOFU/BOFU)
- [ ] Specific campaigns/activities named

**Red Flags:**
- "Do everything" approach
- No prioritization
- Missing budget allocation
- Tactics not linked to strategy
- Generic activities without specifics

---

### Phase 5: Action (Score X/10)

**Must Have:**
- [ ] Timeline with milestones
- [ ] Clear ownership (RACI or similar)
- [ ] Sprint/phase breakdown
- [ ] Resource requirements specified
- [ ] Budget allocated by time period
- [ ] Risk mitigation strategies

**Red Flags:**
- No timeline
- "We'll figure it out" approach
- Missing resource needs
- No risk consideration
- Unassigned responsibilities

---

### Phase 6: Control (Score X/10)

**Must Have:**
- [ ] KPIs linked to each objective
- [ ] Measurement tools specified
- [ ] Review cadence defined (weekly/monthly/quarterly)
- [ ] Success/fail criteria for pivots
- [ ] Attribution model selected
- [ ] Dashboard/reporting structure

**Red Flags:**
- "We'll track everything"
- No measurement tools
- Missing review rhythm
- No pivot triggers
- KPIs not linked to objectives

---

## Cross-Phase Consistency Audit

### Situation → Objectives
- [ ] Objectives address identified SWOT gaps
- [ ] Targets account for market size constraints
- [ ] Competitor insights inform differentiation goals

### Objectives → Strategy
- [ ] Strategy segments can realistically reach targets
- [ ] Strategic pillars directly support objectives
- [ ] North Star Metric is strategy-aligned

### Strategy → Tactics
- [ ] Tactics serve the defined positioning
- [ ] Channel mix reaches target segments
- [ ] Activities deliver on value proposition

### Tactics → Action
- [ ] Timeline is realistic for tactic scope
- [ ] Resources match tactic requirements
- [ ] Budget covers prioritized activities

### Action → Control
- [ ] KPIs measure every major activity
- [ ] Review rhythm matches action cadence
- [ ] Pivot triggers tied to action milestones

---

## Industry Standard Checklist

### Evidence-Based
- [ ] All major claims have sources
- [ ] Benchmarks cited for targets
- [ ] Competitor claims verified

### Measurable
- [ ] Every objective has a number
- [ ] Every tactic has a KPI
- [ ] Every action has a deadline

### Actionable
- [ ] Someone can execute from this document
- [ ] Dependencies are clear
- [ ] Next 30 days are explicit

### Realistic
- [ ] Budget matches ambitions
- [ ] Timeline accounts for capacity
- [ ] Resource constraints acknowledged

### Coherent
- [ ] Strategy explains how objectives are achieved
- [ ] Tactics deliver the strategy
- [ ] Control measures what matters

## Quality Thresholds

| Score | Status | Action |
|-------|--------|--------|
| 50-60 | Excellent | Approve and execute |
| 40-49 | Good | Minor revisions, then approve |
| 30-39 | Adequate | Address gaps before execution |
| Below 30 | Needs Work | Major revision required |

---

## Common Quality Issues & Fixes

| Issue | Symptom | Fix |
|-------|---------|-----|
| Assumption-as-fact | "We know customers want..." | Add research source or validate |
| Strategy-tactic confusion | "Our strategy is content marketing" | Content marketing is a tactic; strategy is WHY |
| Missing funnel logic | All tactics are TOFU | Balance TOFU/MOFU/BOFU |
| Vanity metrics | "Grow followers" | Link to business outcome |
| No pivot triggers | "We'll see how it goes" | Define specific thresholds |
