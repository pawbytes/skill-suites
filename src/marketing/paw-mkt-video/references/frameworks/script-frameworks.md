# Script Frameworks

### Framework 1: Hook-Story-Offer (HSO)
Best for: Sales videos, product content, conversion-focused videos.

**Structure**:
1. **Hook** (0-5s): Grab attention with one of the formulas above
2. **Story** (5s-80%): Tell a relatable story that illustrates the problem and journey
3. **Offer** (final 20%): Present your solution/product as the natural resolution

**Template**:
```
[HOOK]: "I was spending 4 hours a day on [task] until I discovered this."
[STORY]: "Three months ago, I was [struggling with problem]. I tried [common solution 1], [common solution 2], nothing worked. Then I found [discovery]..."
[TRANSITION]: "Here's what changed everything..."
[OFFER]: "That's why I created [product/solution]. It does [benefit 1], [benefit 2], and [benefit 3]. [CTA]."
```

### Framework 2: PAS (Problem-Agitate-Solve)
Best for: Educational content, product demos, pain-point content.

**Structure**:
1. **Problem** (0-10s): State the problem clearly and specifically
2. **Agitate** (10s-40%): Make the problem feel urgent and painful
3. **Solve** (40%-end): Present the solution step by step

**Template**:
```
[PROBLEM]: "Most [audience] waste [time/money] on [ineffective thing]."
[AGITATE]: "And the worst part? [Consequence 1]. [Consequence 2]. Meanwhile, [competitor/alternative] is [getting ahead]."
[SOLVE]: "Here's the 3-step fix. Step 1: [action]. Step 2: [action]. Step 3: [action]. The result? [Outcome]."
```

### Framework 3: AIDA for Video
Best for: Ads, landing page videos, conversion content.

**Structure**:
1. **Attention** (0-3s): Hook that stops the scroll
2. **Interest** (3-15s): Why this matters to the viewer
3. **Desire** (15s-70%): Show the transformation or result
4. **Action** (final 30%): Clear CTA

**Template**:
```
[ATTENTION]: "Stop scrolling if you [audience identifier]."
[INTEREST]: "I found a way to [desirable outcome] without [common pain]."
[DESIRE]: "Imagine [future state]. [Proof point]. [Social proof]. [Demonstration]."
[ACTION]: "[Specific CTA]. Link in bio / Click below / Comment [word] for [offer]."
```

### Framework 4: List Framework
Best for: Tips, hacks, tools, mistakes-to-avoid content.

**Structure**:
1. **Hook + Promise** (0-5s): "X [tips/mistakes/tools] that will [outcome]"
2. **Items** (5s-90%): Deliver each item with a mini-hook, explanation, and example
3. **Bonus + CTA** (final 10%): Extra tip + call to action

**Retention technique**: Tease the best item. "Save #[last] for the end -- it's the most important."

### Framework 5: Before-After-Bridge (BAB)
Best for: Transformations, case studies, testimonial content.

**Structure**:
1. **Before** (0-20%): Show the "before" state (the problem, the struggle)
2. **After** (20-50%): Show the "after" state (the result, the transformation)
3. **Bridge** (50-end): Explain exactly how to get from before to after

**Template**:
```
[BEFORE]: "6 months ago, our [metric] was [bad number]."
[AFTER]: "Today, it's [impressive number]."
[BRIDGE]: "Here are the 3 exact changes we made. Change 1: [explanation]. Change 2: [explanation]. Change 3: [explanation]."
```
