# Video SEO Checklist

### Title Optimization
- [ ] Primary keyword in first 5-6 words
- [ ] Under 60 characters (avoids truncation)
- [ ] Includes a hook element (number, question, bold claim)
- [ ] Matches search intent (informational, transactional, navigational)
- [ ] Does NOT keyword-stuff; reads naturally
- [ ] Complements thumbnail (together they tell the full story)

### Description Optimization
- [ ] Primary keyword in first 1-2 sentences
- [ ] First 150 characters are compelling (shown in search results)
- [ ] Include 2-3 relevant secondary keywords naturally
- [ ] Add timestamps/chapters (improves SEO and UX)
- [ ] Include relevant links (website, socials, mentioned resources)
- [ ] Add a brief summary paragraph (100-200 words)
- [ ] Include a CTA in the description

### Tags & Hashtags
- [ ] Primary keyword as first tag
- [ ] 5-10 relevant tags (mix of broad and specific)
- [ ] Include brand name as a tag
- [ ] Include common misspellings of key terms
- [ ] 3-5 hashtags in description (shown above title on YouTube)

### Chapters (Timestamps)
- [ ] Add timestamps for all major sections
- [ ] First timestamp starts at 0:00
- [ ] Minimum 3 chapters, maximum 10-15
- [ ] Chapter titles include relevant keywords
- [ ] Format: `0:00 Introduction` / `2:15 Step 1: [Keyword-rich title]`

### Cards & End Screens
- [ ] Add 1-3 cards at relevant moments (link to related content)
- [ ] Place cards when viewer might search for more info (not at hook)
- [ ] End screen (last 20 seconds): Subscribe button + best next video
- [ ] End screen video: Choose your best-performing related video
- [ ] Script the end screen: "If you enjoyed this, watch THIS next..."

### Closed Captions / Subtitles
- [ ] Upload accurate captions (auto-generated are 80-90% accurate)
- [ ] Review and fix auto-captions for key terms and brand names
- [ ] Consider adding captions for additional languages (top viewer demographics)
- [ ] Captions are indexed by YouTube for search (SEO benefit)

### Engagement Optimization
- [ ] Pin a comment with a question or CTA
- [ ] Reply to comments within first 1-2 hours (boosts algorithm)
- [ ] Ask a question in the video to encourage comments
- [ ] Include a verbal subscribe CTA mid-video (not just intro/outro)
