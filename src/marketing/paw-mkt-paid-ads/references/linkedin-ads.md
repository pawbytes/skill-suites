# LinkedIn Ads Strategy

B2B-focused advertising guidance for LinkedIn campaigns. For extended best practices on Document Ads and Thought Leader Ads, see `./references/best-practices.md`.

---

## 1. Campaign Types

### Sponsored Content
- Single image, carousel, video, document ads in feed
- Primary B2B format

### Message Ads
- Direct InMail with high open rates (40-50%)
- Use for high-value offers, event invites, demo requests
- Frequency cap: 1 message per member per 45 days

### Conversation Ads
- Choose-your-own-path messages
- Multiple CTAs, good for qualifying leads

### Text Ads
- Right rail and top banner
- Low cost, low CTR - brand awareness filler

### Lead Gen Forms
- Pre-filled from profile data
- 2-3x higher conversion than landing pages
- Keep to 3-4 fields max

---

## 2. B2B Targeting

### Job Title / Function / Seniority
- Target decision-makers directly
- Combine function + seniority for precision

### Company Size / Industry / Name
- ABM by uploading company lists
- Layer company size for SMB vs enterprise campaigns

### Skills / Groups / Interests
- Reach niche professionals by skill endorsements or group membership

### Matched Audiences
- Website retargeting
- Contact list upload
- Lookalikes
- ABM company lists

### Minimum Audience Size
- 50,000+ for sponsored content
- 15,000+ for message ads

---

## 3. ABM (Account-Based Marketing)

1. Upload target account lists (CSV)
2. Create matched audience
3. Layer with seniority and function to reach decision-makers at target companies
4. Run awareness before lead gen
5. Coordinate with sales on account prioritization

## 4. Revenue Attribution for Long B2B Sales Cycles

**Challenge:** B2B sales cycles are 3-12 months. Platform attribution windows (90 days max) miss most downstream revenue.

**Solutions:**
- Pass LinkedIn Click ID (li_fat_id) to CRM via hidden form fields
- Match CRM closed-won deals back to LinkedIn campaigns manually or via integration
- Use LinkedIn Revenue Attribution Report
- Implement MMM (Marketing Mix Modeling) for holistic channel contribution
- Track leading indicators: MQL rate, SQL rate, pipeline value generated
