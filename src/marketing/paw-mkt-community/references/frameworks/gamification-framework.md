# Gamification Framework

## Points System

| Action | Points | Notes |
|---|---|---|
| Post a discussion | 10 | Original content creation |
| Reply to a discussion | 5 | Conversation contribution |
| Receive a reaction/like | 2 | Quality signal from peers |
| Complete profile | 15 | One-time onboarding action |
| Post an introduction | 20 | One-time activation action |
| Attend an event | 25 | Live participation |
| Host or co-host an event | 50 | Leadership action |
| Refer a new member | 30 | Growth contribution |
| Complete a challenge | 40 | Sustained engagement |
| Create a resource or guide | 50 | High-value content contribution |
| Get featured in member spotlight | 75 | Recognition milestone |

## Level System

| Level | Name | Points Required | Unlocked Perks |
|---|---|---|---|
| 1 | Newcomer | 0 | Basic community access |
| 2 | Member | 50 | Access to resource library, ability to create polls |
| 3 | Contributor | 200 | Exclusive channel/space access, ability to host threads |
| 4 | Expert | 500 | Direct team access channel, early feature access, custom badge |
| 5 | Elder | 1,000 | Advisory board eligibility, co-creation invitations, annual summit invite |
| 6 | Legend | 2,500 | Permanent recognition, lifetime benefits, ambassador invitation |

## Badges

| Badge | Criteria | Category |
|---|---|---|
| Founding Member | Joined during founding period | Identity |
| First Post | Made first discussion post | Activation |
| Conversation Starter | Started 10 discussions that each received 3+ replies | Engagement |
| Helpful Hand | Received 25 "helpful" reactions on answers | Quality |
| Event Regular | Attended 5+ community events | Participation |
| Event Host | Hosted or co-hosted a community event | Leadership |
| Challenge Champion | Completed 3+ community challenges | Commitment |
| Recruiter | Referred 5+ new members | Growth |
| Content Creator | Published 10+ original resources or guides | Contribution |
| Milestone Member | Active for 12+ consecutive months | Loyalty |

## Leaderboard Design

- **Monthly reset**: Leaderboards reset each month to keep competition accessible. New members can compete immediately.
- **Top 3 recognition**: Monthly recognition for the top 3 members. Prize options: exclusive access, swag, 1:1 call with team, feature in community spotlight.
- **Category leaderboards**: Separate leaderboards for different activities (most helpful, most active, most referrals) so different behavior types are rewarded.
- **Opt-out option**: Allow members to opt out of public leaderboard if they prefer privacy.

**Important caveat**: Gamification amplifies existing engagement. It cannot fix a community with no inherent value. If members are not engaged without points, points will not save it.
