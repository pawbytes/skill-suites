## Codec Quick Reference

| Container | Video Codec | Audio Codec | Profile | Pixel Format |
|-----------|-------------|-------------|---------|--------------|
| MP4 | H.264 (libx264) | AAC (aac) | High | yuv420p |

All platforms accept H.264 in MP4 container. This is the universal safe choice. No platform currently requires H.265/HEVC, VP9, or AV1 for uploads.

**Recommended ffmpeg encoding flags:**
```
-c:v libx264 -profile:v high -pix_fmt yuv420p -crf 20 -preset medium -c:a aac -b:a 128k -movflags +faststart
```

The `-movflags +faststart` flag moves the moov atom to the beginning of the file for faster web playback. Always include it.
