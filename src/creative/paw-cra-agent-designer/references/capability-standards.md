# Capability Quality Standards

Reference for what constitutes production-ready output. These standards define the quality bar—what separates professional work from amateur.

---

## Universal Standards

### Technical Quality Checklist

- [ ] Correct dimensions (no approximations)
- [ ] Correct color mode (RGB digital, CMYK print)
- [ ] Resolution appropriate (72-150 PPI digital, 300 DPI print)
- [ ] File format correct for use case
- [ ] File size optimized (not bloated)

### Design Quality Checklist

- [ ] Clear visual hierarchy (headline → supporting → CTA)
- [ ] Text legible at intended viewing size
- [ ] Adequate white space (30%+ recommended)
- [ ] Composition balanced and intentional
- [ ] Message immediately understood

### Brand Alignment Checklist

- [ ] Colors match brand guidelines exactly
- [ ] Typography follows brand specs
- [ ] Logo used correctly (proper version, clear space, minimum size)
- [ ] Visual style consistent with brand identity

## Social Post Standards

**Technical:**
- Exact platform dimensions (reference `platform-specifications.md`)
- 24pt minimum body text for mobile
- Safe zones: 60px from edges (Instagram), 50px (LinkedIn)
- Export: JPG (photos), PNG (graphics with transparency)

**Quality Indicators:**

| High Quality | Low Quality |
|--------------|-------------|
| Immediate message clarity | Unclear purpose |
| Strong visual hierarchy | Competing elements |
| On-brand aesthetic | Generic or off-brand |
| Mobile-optimized text | Text too small for mobile |

**Platform Notes:**
- **Instagram:** Portrait (4:5) gets 25% more screen space than square
- **LinkedIn:** Professional aesthetic, PDF carousels max 10 slides
- **TikTok:** 150px safe zone at bottom for UI
- **X/Twitter:** 16:9 displays best in feed

## Carousel Standards

**Technical:**
- All slides identical aspect ratio (1:1 or 4:5)
- 1080px minimum width
- Max 20 slides (Instagram), 10 slides (LinkedIn PDF)
- Sequential numbering or progress indicators

**Structure Standards:**

**First Slide (Hook):**
- Attention-grabbing headline
- High contrast for scroll-stopping
- Clear value proposition (<12 words)
- Visual cue to swipe (arrow, "swipe →")

**Interior Slides:**
- One key idea per slide
- Consistent typography/color
- Progressive information reveal
- Strategic CTA placement (not just final slide)

**Final Slide:**
- Clear call-to-action
- Contact/handle information
- Branding reinforcement

**Optimal Slide Count:**

| Slides | Best For |
|--------|----------|
| 5-7 | Quick tips, simple concepts |
| 8-10 | Tutorials, detailed explanations |
| 10-20 | Comprehensive guides |

## Flyer Standards

**Technical:**

| Output | Specifications |
|--------|---------------|
| **Digital** | 1080px+ width, RGB, JPG/PNG |
| **Print** | 300 DPI, CMYK, PDF with bleed |

**Print Margins:**
- Bleed: 3mm (0.125") beyond trim on all sides
- Safety zone: 3mm inside trim
- Critical content inside safety zone only

**Visual Hierarchy Order:**
1. Headline — Largest, boldest
2. Key visual — Supporting image
3. Key information — Date, time, location, price
4. Details — Additional information
5. Call-to-action — What to do next
6. Contact/Branding — Smallest, but present

**Composition Rules:**
- Clear focal point
- Balanced layout
- 30%+ white space
- Logical reading order

## Brand Asset Standards

**Logo Requirements:**

| Requirement | Standard |
|-------------|----------|
| **Formats** | PNG (transparent), SVG (vector), JPG (white bg) |
| **Variations** | Full color, monochrome, reversed (white) |
| **Sizes** | 16px, 32px, 64px, 256px, 512px, 1024px |
| **Scalability** | Works at business card AND billboard size |

**Logo Design Principles:**
1. Simple — Recognizable at any size
2. Memorable — Distinctive and unique
3. Timeless — Won't look dated in 5 years
4. Versatile — Works across applications
5. Appropriate — Fits the brand/industry

**Clear Space Rule:**
- Maintain empty space around logo equal to height of a defined element (often the first letter)
- No other elements within this zone

**Minimum Sizes:**
- Print: 0.5" width minimum
- Digital: 24px height minimum

**Icon Standards:**

| Requirement | Standard |
|-------------|----------|
| **Format** | SVG (preferred), PNG at multiple sizes |
| **Style** | Consistent stroke weight, corner radius |
| **Grid** | Pixel-aligned for crisp rendering |
| **Sizes** | 16, 24, 32, 48, 64, 128px |

## Asset Resizing Standards

**Resize Strategies:**

| Scenario | Strategy |
|----------|----------|
| Same aspect ratio | Direct scale |
| Landscape to portrait | Crop to focal point, extend background |
| Portrait to landscape | Extend sides, restack elements |
| Square to vertical | Extend top/bottom |
| Print to digital | Adjust resolution, CMYK→RGB |

**Quality Preservation:**
- Never upscale beyond 150% without AI enhancement
- Check for artifacts after resize
- Verify text remains legible
- Confirm brand elements intact

## Accessibility Standards

- Color contrast: 4.5:1 minimum for text
- Text not too small (24pt+ for mobile body)
- Information not conveyed by color alone
- Alt text prepared for digital images
