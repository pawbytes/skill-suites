# Campaign Template: Product Launch

Activated with `--template:product-launch`. This template pre-populates a standard deliverable set for product launches, merged from the legacy `cra-product-launch-campaign` workflow.

## Standard Product Launch Deliverable Set

When this template is active, merge these standard deliverables with whatever the brief specifies. Skip any that the brief explicitly excludes.

### Pre-Launch Phase
- **Teaser post** (Designer, social-post) — mysterious/intriguing visual for Instagram, X, LinkedIn
- **Countdown carousel** (Designer, carousel) — 3-5 slide countdown series
- **Teaser video** (Video Producer, short-video) — 15-30 second vertical teaser for Reels/TikTok/Shorts

### Launch Day Phase
- **Announcement post** (Designer, social-post) — hero image for all platforms
- **Feature carousel** (Designer, carousel) — product features walkthrough, 5-8 slides
- **Launch video** (Video Producer, short-video) — 30-60 second product reveal
- **Demo video** (Video Producer, long-video) — 2-5 minute product demo for YouTube/web

### Post-Launch Phase
- **Social proof post** (Designer, social-post) — testimonial or stat highlight
- **Behind-the-scenes video** (Video Producer, short-video) — team/making-of content
- **Recap carousel** (Designer, carousel) — launch results or key highlights

## Template Behavior

1. Present the standard set alongside brief-specific deliverables
2. If not headless, let the user add/remove/modify before dispatch
3. If headless, include all standard deliverables plus brief-specific ones
4. Apply campaign deadlines across phases (pre-launch, launch day, post-launch)
5. Tag each deliverable with its phase for scheduling context
