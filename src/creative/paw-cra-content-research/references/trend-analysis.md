# Trend Analysis

Identify trending formats, topics, aesthetics, and platform-specific patterns relevant to the brand's industry and audience.

## Trend Classification

Classify every identified trend before recommending action:

| Category | Lifespan | Content Action |
|----------|----------|----------------|
| **Fad** | <3 months | Quick-turn content. Jump on now or skip entirely. Do not build a series around it. |
| **Trend** | 6-18 months | Build a content series. Experiment with the format. Iterate based on performance. |
| **Movement** | 2+ years | Build brand authority. Create cornerstone content. Position as thought leader. |
| **Declining** | Past peak | Avoid unless using ironically or as a contrast ("remember when everyone did X?"). |

## Research Dimensions

### Content Format Trends
- Carousel styles gaining traction (educational, storytelling, comparison)
- Video formats performing well (green screen, duet/stitch, day-in-life, GRWM)
- Interactive formats (polls, quizzes, Q&A)
- Audio trends (trending sounds, voiceover styles, ASMR)

### Visual & Aesthetic Trends
- Color palette shifts (warm earth tones, neon, muted pastels)
- Typography trends (bold sans-serif, retro serif, handwritten)
- Photography style (raw/unfiltered vs. polished, AI-generated imagery acceptance)
- Layout trends (asymmetric, magazine-style, text-heavy)

### Topical Trends
- What industry topics are generating engagement?
- What questions are audiences asking?
- What cultural moments can the brand align with?
- What seasonal or cyclical trends are approaching?

### Platform-Specific Trends
- **Instagram:** Feature adoption (Collabs, Notes, broadcast channels), algorithm signals
- **TikTok:** Sound trends, effect trends, content pillars gaining reach
- **YouTube:** Format trends (Shorts strategy, long-form hooks, community posts)
- **LinkedIn:** Content format shifts, algorithm preferences, B2B trends

## Production Translation

For every trend identified, answer:
- **How does this translate to a visual asset?** (Designer brief direction)
- **How does this translate to video?** (Video Producer brief direction)
- **Which platform should we execute on first?**
- **What's the minimum viable version?** (fastest path to testing the trend)
