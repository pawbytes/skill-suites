# Production Recommendations

Translate research findings into specific briefs for Designer and Video Producer. Every recommendation must be detailed enough that a production agent can start work without further research.

## Design Brief Format

For each visual content recommendation:

```markdown
### [Angle Name] — Design Brief

**Format:** [carousel / single post / flyer / story / infographic]
**Platform:** [Instagram / TikTok / LinkedIn / multi-platform]
**Dimensions:** [1080x1350 / 1080x1920 / etc.]

**Visual Concept:**
[2-3 sentence description of the visual approach. Be specific about style — "minimalist with bold sans-serif typography overlay on muted photography" not "clean and modern."]

**Reference Style:**
[Name specific visual references: "like Glossier's before/after carousels" or "competitor X's bold quote cards but with our brand palette"]

**Copy Direction:**
[Headline/hook suggestion, tone notes, CTA direction]

**Brand Alignment:**
[Which brand guidelines elements to emphasize — colors, fonts, voice]
```

## Video Brief Format

For each video content recommendation:

```markdown
### [Angle Name] — Video Brief

**Format:** [reel / short / long-form / motion graphic]
**Platform:** [TikTok / Instagram Reels / YouTube Shorts / YouTube]
**Duration:** [15s / 30s / 60s / 3-5min]
**Aspect Ratio:** [9:16 / 16:9 / 1:1]

**Hook (first 3 seconds):**
[Specific hook concept — what the viewer sees and hears immediately]

**Scene Structure:**
1. [Scene description + duration]
2. [Scene description + duration]
3. [Scene description + duration]

**Audio Direction:**
[Music style, voiceover notes, trending sound suggestion if applicable]

**Subtitle Style:**
[Animated word-by-word / static captions / stylized]

**Visual Style:**
[Shooting style reference — talking head, B-roll montage, screen recording, motion graphics]

**Brand Alignment:**
[Color overlay, font for supers, logo placement]
```

## Platform Playbook Format

For each target platform, include:

```markdown
### [Platform] Playbook

**Content Mix:** [% educational / % promotional / % entertaining / % community]
**Posting Cadence:** [suggested frequency based on research]
**Best Formats:** [top 2-3 formats for this platform based on trends]

**Hashtag Strategy:**
- Primary (high-volume): [3-5 hashtags]
- Niche (targeted): [3-5 hashtags]
- Branded: [1-2 hashtags]

**Optimal Timing:** [based on audience and platform trends]
**Platform Features to Use:** [specific features like Collab, Stitch, etc.]
```

## Quality Check

Before finalizing recommendations:
- Does every brief specify dimensions/format/platform? (Production agents need specs, not vague direction)
- Is the visual concept specific enough to generate from? ("bold typography" is vague; "white knockout text on dark gradient with brand accent color underline" is actionable)
- Does the video brief have a hook? (First 3 seconds determine success)
- Are hashtag recommendations current? (Stale hashtags signal inauthenticity)
- Does every recommendation trace back to a research finding? (No untethered ideas)
