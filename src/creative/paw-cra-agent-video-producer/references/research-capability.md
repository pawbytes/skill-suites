# Research & Learn

## Outcome

Research video production techniques, platform updates, encoding best practices, and trending video formats -- then save findings to the shared knowledge base for future use.

## Trigger

User asks to research video techniques, learn about a platform's video specs, explore trending video styles, or investigate a technical video topic.

## Knowledge Base

Check existing knowledge first: `{project-root}/.pawbytes/creative-suites/knowledge/index.md`

If the topic has already been researched, load the existing entry. Only conduct new research if the topic is not covered or if the user requests updated information.

## Research Topics

Common research areas for video production:

- **Platform updates** -- new video specs, algorithm changes, feature launches
- **Encoding techniques** -- codec advances, compression strategies, quality optimization
- **Trending formats** -- popular video styles, viral patterns, emerging content types
- **Tool updates** -- new fal.ai models, ffmpeg features, Remotion capabilities
- **Production techniques** -- transition trends, subtitle styles, audio mixing approaches
- **AI video advances** -- new generation models, quality improvements, prompt techniques

## Process

1. Check knowledge base for existing coverage
2. If not found or stale, research the topic using available web search tools
3. Synthesize findings into a concise, actionable reference
4. Save to knowledge base at `.pawbytes/creative-suites/knowledge/`
5. Update `knowledge/index.md` with new entry

## Output

Research findings saved to knowledge base for future reference by all agents in the suite. Present key findings to the user with actionable recommendations for their current production needs.
