# Copy Drafting

Generate platform-optimized copy using proven persuasion formulas.

**Key Methodologies:** AIDA, PAS, BAB, 4 C's. Load `./methodologies.md` for complete formula details and examples.

## Choosing the Right Formula

| Situation | Formula | Why |
|-----------|---------|-----|
| Landing pages, sales pages | AIDA | Guides through full funnel |
| Problem-aware audiences | PAS | Agitates pain before solution |
| Case studies, transformations | BAB | Shows before/after contrast |
| Value propositions, descriptions | 4 C's | Ensures clarity and credibility |

## Formula Quick Reference

### AIDA (Classic Funnel)
1. **Attention:** Compelling headline that grabs
2. **Interest:** Build engagement with facts/stories
3. **Desire:** Paint the transformation
4. **Action:** Clear, direct CTA

### PAS (Problem-Agitate-Solution)
1. **Problem:** State the core problem clearly
2. **Agitate:** Emotionally amplify the pain
3. **Solution:** Present as the obvious answer

### BAB (Before-After-Bridge)
1. **Before:** Paint the painful current state
2. **After:** Paint the transformed future
3. **Bridge:** Explain how you get them there

### 4 C's (Clarity + Connection)
- **Clear:** Immediately understandable?
- **Concise:** Every word necessary?
- **Compelling:** Creates desire?
- **Credible:** Backed by proof?

## Input

Gather from conversation or campaign brief:
- Platform (Instagram, TikTok, LinkedIn, Twitter/X, email, ad network)
- Content type (post, ad, email, caption, headline)
- Key message or offer
- Call to action
- Tone (professional, playful, urgent, inspirational)
- Any constraints (character limits, compliance requirements)

## Platform Considerations

| Platform | Optimal Length | Hashtags | CTAs |
|----------|----------------|----------|------|
| Instagram | 125-150 chars | 5-15 relevant | Link in bio, Save, Share |
| TikTok | 100-150 chars | 3-5 trending | Follow, Check profile |
| LinkedIn | 150-300 chars | 3-5 industry | Comment, Share |
| Twitter/X | 100-280 chars | 1-2 | Retweet, Reply |
| Email | 50-200 words | None | Click, Reply |
| Ad | 90 chars primary | None | Click, Learn More |

## Copy Elements

Generate variations for each element:

1. **Headlines/Hooks** (3-5 options)
2. **Body copy** (2-3 variations)
3. **Call to action** (2-3 options)
4. **Hashtags** (platform-appropriate set)

## Output

Create `{campaign}/copy/{platform}-{type}.md`:

```markdown
# Copy: {Platform} - {Type}
## Metadata
- Platform: {platform}
- Type: {post/ad/email}
- Campaign: {campaign name}
- Tone: {tone}

## Headlines
1. "{headline option 1}"
2. "{headline option 2}"
3. "{headline option 3}"

## Body Copy Options

### Option A
{full body copy}

### Option B
{full body copy}

## CTAs
1. {CTA option 1}
2. {CTA option 2}

## Hashtags
{formatted hashtag block for platform}

## Compliance Notes
- {Any required disclosures}
- {Restricted claims to avoid}
```

## Completion Criteria

Copy is production-ready when:
- [ ] Clear, scannable copy (key message front-loaded)
- [ ] Strong hook in first line
- [ ] Single, clear CTA
- [ ] Platform-native voice and length
- [ ] At least 2 headline variations provided
- [ ] Compliance requirements met (disclosures, restricted claims)

## Quality Standard

- Clear, scannable copy (front-load key message)
- Strong hook in first line
- Single, clear CTA
- Platform-native voice
- Test variations provided
- Compliance requirements met
