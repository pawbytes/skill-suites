# Content Calendar

Plan content schedule with themes, timing, and specifications using the Content Pillar framework.

**Key Methodologies:** Content Pillar Strategy, Pillar/Cluster Structure. Load `./methodologies.md` for complete framework details.

## Content Pillar Strategy

Build topical authority through pillar content supported by cluster content:

- **Pillar Content:** Comprehensive, authoritative content covering a broad topic
- **Cluster Content:** Specific, detailed content covering subtopics
- **Internal Linking:** Clusters link to pillar, pillar links to clusters

**Typical Structure:** 3-5 pillars, each with 5-10 cluster pieces

## Input

Gather from conversation or campaign brief:
- Campaign duration
- Platforms to publish on
- Posting frequency goal
- Content pillars/themes
- Key dates (launches, events, holidays)
- Content types (video, static, carousel, stories)

## Calendar Structure

Build a calendar that includes:

1. **Posting schedule** (day/time by platform)
2. **Content themes** (pillars that rotate)
3. **Content types** (format per post)
4. **Hooks/topics** (specific content ideas)
5. **Asset requirements** (what needs to be created)

## Platform Timing Best Practices

| Platform | Best Days | Best Times (local) |
|----------|-----------|-------------------|
| Instagram | Tue, Wed, Fri | 11am, 2pm, 7pm |
| TikTok | Tue, Wed, Thu | 7am, 10am, 7pm |
| LinkedIn | Tue, Wed, Thu | 8am, 12pm |
| Twitter/X | Daily | 8am, 12pm, 5pm |
| YouTube | Thu, Fri, Sat | 2pm, 5pm |

*Adjust based on audience insights when available.*

## Completion Criteria

Calendar is complete when:
- [ ] Realistic posting schedule defined (sustainable cadence)
- [ ] Content pillars established with descriptions
- [ ] Variety in content types across platforms
- [ ] Key dates identified (events, launches, holidays)
- [ ] Asset requirements with due dates specified
- [ ] Cross-promotion strategy noted

## Quality Standard

- Realistic posting schedule (sustainable cadence)
- Variety in content types
- Clear asset handoff to Designer/Video Producer
- Key dates identified
- Themes that build narrative over time
