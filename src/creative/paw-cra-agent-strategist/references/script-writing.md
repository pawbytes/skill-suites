# Script Writing

Write scripts for videos, podcasts, and multimedia content using the 7-Part Retention Framework.

**Key Methodologies:** 7-Part Retention Framework, Hook Formulas, Pattern Interrupt Techniques, Value Bomb Placement. Load `./methodologies.md` for complete framework details.

## Core Principle

YouTube rewards percentage watched, not total watch time. Structure scripts to maintain engagement throughout.

## The 7-Part Framework

| Part | Timing | Purpose |
|------|--------|---------|
| **1. Hook** | 0-8 sec | Pattern interrupt, break scroll momentum |
| **2. Anti-Hook** | 8-15 sec | Acknowledge skepticism, build trust |
| **3. Promise** | 15-30 sec | Specific outcome + capability |
| **4. Preview + Open Loops** | 30-60 sec | Create curiosity gaps |
| **5. Core Content** | 60-80% | Deliver value with pattern interrupts every 45-90 sec |
| **6. Value Bomb** | 60-70% mark | Single best insight placed before drop-off |
| **7. Soft CTA** | Final 10% | Natural next step |

## Hook Formulas

- **Contradiction:** "Everyone says [common advice]. But the data shows [opposite]."
- **Specific Number:** "After analyzing [number] [things], I found [insight]."
- **Identity:** "If you're a [identity] who struggles with [problem], this changes everything."
- **Mistake:** "I made this mistake for [time]. It cost me [consequence]."

## Pattern Interrupts (use every 45-90 seconds)

- "Here's what most people miss..." — signals insider knowledge
- 5-10 second personal story — humanizes content
- Direct address — "Now, you might be thinking..."
- Visual change — different angle, location, b-roll
- Energy shift — speed up or slow down deliberately

## Input

Gather from conversation or campaign brief:
- Content type (video, podcast, ad, explainer)
- Duration or format (15s, 60s, 3min, long-form)
- Platform (TikTok, YouTube, Instagram, podcast)
- Topic and key messages
- Tone (educational, entertaining, inspirational, promotional)
- Call to action

## Script Structure by Format

### Short-Form Video (15-60s)
```
HOOK (0-3s): {attention-grabbing opening}
CONTEXT (3-10s): {set up the premise}
VALUE (10-45s): {main content, key points}
CTA (last 3-5s): {specific call to action}
```

### Long-Form Video (2-10min)
```
HOOK (0-30s): {compelling reason to watch}
INTRO (30s-1min): {context, what you'll learn}
SECTION 1: {first major point}
SECTION 2: {second major point}
SECTION 3: {third major point}
RECAP: {quick summary}
CTA: {specific call to action}
```

### Podcast Segment
```
TEASER: {intriguing preview}
INTRO: {host intro, topic setup}
DISCUSSION: {key points, stories, insights}
TRANSITIONS: {natural segues between topics}
OUTRO: {summary, next steps}
```

### Advertisement
```
PROBLEM: {identify pain point}
SOLUTION: {introduce product/service}
PROOF: {evidence, testimonial, demo}
CTA: {specific action}
```

## Output

Create `{campaign}/scripts/{script-name}.md`:

```markdown
# Script: {Title}
## Metadata
- Format: {format}
- Duration: {target}
- Platform: {platform}
- Tone: {tone}

## Script

### Scene 1: {Scene Name}
**Visual:** {description of what's on screen}
**Audio:** {voiceover, music, sound effects}
**Text overlay:** {any on-screen text}

**VO:** "{exact spoken words}"

### Scene 2: {Scene Name}
...

## Production Notes
- {Any specific direction for production}
- {Props, locations, or assets needed}

## Alternative Hooks
1. "{alternative hook option 1}"
2. "{alternative hook option 2}"
```

## Completion Criteria

Script is production-ready when:
- [ ] Hook grabs attention in first 3 seconds
- [ ] Structure follows appropriate framework for format
- [ ] Natural, conversational language (read aloud test passed)
- [ ] Visual direction provided for each scene
- [ ] Clear call to action
- [ ] Alternative hooks provided for testing

## Quality Standard

- Hook must grab attention in first 3 seconds
- Every line serves the narrative
- Natural, conversational language (read it aloud)
- Clear call to action
- Include visual direction for production team
- Provide alternatives for testing
