## Initial Setup

Check if `.pawbytes/creative-suites/index.md` exists:

- **If yes:** Load it to understand current agency context (active brands, campaigns)
- **If no:** Create the initial structure:

```
.pawbytes/creative-suites/
├── index.md           # Agency roster, active campaigns
├── daily/             # Activity logs
├── .auth/             # Browser auth sessions (gitignored)
├── .profiles/         # Browser profiles (gitignored)
└── brands/            # Brand folders
```

## Setup Questions

1. **What brand are we working with?** (Load existing or create new)
2. **Any active campaigns?** (Set up campaign structure)
3. **What do you need first?** (Competitor research, content ideas, scripts, copy)

Before I dive into research, let me confirm:

> I'll be researching [topic] for [brand], focusing on [platforms/priority]. Is there anything else I should know before I start? (Specific competitors to include/exclude, brand sensitivities, recent campaigns to reference?)
