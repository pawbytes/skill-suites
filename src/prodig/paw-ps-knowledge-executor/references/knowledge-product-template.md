# Knowledge Products

Guidance for creating knowledge products including courses, ebooks, guides, and memberships.

## Product Type Overview

| Type | Primary Outcome | Format | Commitment |
|------|-----------------|--------|------------|
| Course | Skill acquisition | Structured lessons | 4-12 hours |
| Ebook | Knowledge transfer | Written chapters | 1-4 hours reading |
| Guide | Problem solving | Action-oriented sections | 30min-2 hours |
| Membership | Ongoing value | Recurring content | Monthly |

## Universal Principles

All knowledge products follow these core principles:

### 1. Transformation First

Every product delivers a specific transformation:
- Clear before/after state
- Measurable outcome
- Customer can articulate what they achieve

### 2. Structure for Learning

Content is organized for comprehension:
- Logical progression
- Appropriate chunking
- Regular summaries and checkpoints

### 3. Action Orientation

Every section enables doing:
- Exercises and activities
- Templates and tools
- Implementation guidance

### 4. Sellable Quality

Products are ready to sell:
- Professional presentation
- Clear value proposition
- Tangible deliverables

## Quality Standards

### Structure Quality

- [ ] Clear transformation statement
- [ ] Logical progression
- [ ] Consistent formatting
- [ ] Scannable headers and lists
- [ ] Complete (no obvious gaps)

### Content Quality

- [ ] Accurate and current
- [ ] Actionable and specific
- [ ] Appropriate for audience
- [ ] Examples and stories included
- [ ] Key takeaways defined

### Experience Quality

- [ ] Easy to navigate
- [ ] Clear next steps
- [ ] Support resources identified
- [ ] Progress trackable

## Creation Workflow

1. **Define Transformation** — What does the customer achieve?
2. **Design Structure** — How is content organized?
3. **Plan Content** — What goes in each section?
4. **Create Draft** — Write/build core content
5. **Add Assets** — Templates, worksheets, tools
6. **Review Quality** — Check against standards
7. **Package** — Format for delivery

## Common Patterns

### Course Patterns

- Linear progression (beginner to advanced)
- Modular toolkit (mix and match)
- Project-based (build something)
- Cohort-based (community-driven)

### Ebook Patterns

- How-to guide (step-by-step)
- Framework book (methodology)
- Reference guide (lookup)
- Case study collection (examples)

### Membership Patterns

- Training library (content archive)
- Template club (resources)
- Community hub (connection)
- Coaching access (expert help)
