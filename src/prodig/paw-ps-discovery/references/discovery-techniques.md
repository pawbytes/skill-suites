# Discovery Techniques

Creative ideation techniques for expanding possibility spaces and finding new angles.

## Core Techniques

### 1. Dimensional Expansion

Stretch an idea along different dimensions:

| Dimension | Stretch Question | Example |
|-----------|------------------|---------|
| Scale | "What if this was 10x bigger or smaller?" | Course → Learning ecosystem |
| Speed | "What if this was instant?" | Manual process → Automated tool |
| Depth | "What if this went much deeper?" | Tips → Complete methodology |
| Breadth | "What if this covered more ground?" | Single skill → Multi-skill system |
| Audience | "Who else might need this?" | Beginners → Advanced practitioners |

### 2. Problem Inversion

Flip the problem to find new solutions:

- "What's the opposite problem?"
- "What would make this worse?" (then invert)
- "What if the constraint was the feature?"

Example: "How might we help people write more?" inverted becomes "How might we help people need to write less?" — leading to templates, systems, AI assistance.

### 3. Adjacent Possible

Explore what's one step away:

- "What's the natural next step after someone uses this?"
- "What comes before this problem?"
- "What happens alongside this?"

Map the adjacency:

```text
        [Pre-problem]
             |
[Related problem] --- [Core problem] --- [Next problem]
             |
        [Post-solution]
```

### 4. Analogy Mining

Find inspiration from other domains:

- "This is like X for Y. What else is X for?"
- "How is this problem solved in other industries?"
- "What would this look like in [different domain]?"

Example: "Writing consistently is like..." — exercise (workout plans), cooking (meal prep), investing (dollar-cost averaging).

### 5. Constraint Removal

Remove assumed limitations:

- "If money wasn't a factor..."
- "If you had all the time in the world..."
- "If technology could do anything..."
- "If you could only work on this..."

Then bring constraints back selectively.

### 6. Hybridization

Combine ideas to create new possibilities:

- [Idea A] + [Idea B] = ?
- "What if [concept] had [feature from another concept]?"
- "What would [product type] meet [different product type] look like?"

Example: Course + Community = Cohort-based program. Template + Coaching = Done-with-you service.

### 7. Timeline Projection

Explore temporal variations:

- "What's the 5-minute version?"
- "What's the 5-week version?"
- "What's the 5-month version?"
- "What's the forever version?"

### 8. Identity Exploration

Help users see themselves in the product:

- "Who does this help them become?"
- "What identity shift does this enable?"
- "What would someone call themselves after using this?"

## Session Openers

Kick off discovery sessions with energizing prompts:

| Opener | When to Use |
|--------|-------------|
| "What's the most exciting version of this you can imagine?" | User has a seed idea |
| "If you could wave a magic wand, what would exist?" | User feels constrained |
| "What do you wish existed that doesn't?" | User is searching |
| "Tell me about a time you solved this problem creatively." | User has a specific problem |
| "What's the thing nobody's talking about in [domain]?" | User wants differentiation |

## Divergence Prompts

Generate more possibilities:

- "And what else?"
- "What's another way to think about this?"
- "What would [specific person/company] do with this?"
- "What if this had to be fun?"
- "What would make this irresistible?"
- "What's the version you'd be most proud of?"

## Convergence Prompts

Help filter and focus:

- "Which of these excites you most right now?"
- "If you could only do one, which would it be?"
- "What feels most like 'you'?"
- "Which one would you actually finish?"
- "What would make this worth doing?"

## Energy Reading

Pay attention to user energy:

| Signal | Meaning | Response |
|--------|---------|----------|
| Voice/tone lifts | Idea resonates | Explore deeper |
| "Hmm, not quite" | Direction is off | Try another angle |
| "That's interesting..." | Curiosity piqued | Build on it |
| Silence/hesitation | Processing or uncertain | Give space, then clarify |
| Rapid follow-up questions | High engagement | Keep going |

## Anti-Patterns

Avoid these discovery killers:

| Anti-Pattern | Problem | Instead |
|--------------|---------|---------|
| Premature filtering | Cuts off possibilities | Generate first, filter later |
| Executor mindset | Jumps to "how" | Stay in "what if" |
| Judgment language | Discourages wild ideas | "Yes, and..." |
| Single-option fixation | Misses alternatives | "What else?" |
| Over-structuring | Kills creative flow | Loosen the framework |

## Technique Selection Guide

| Situation | Start With |
|-----------|------------|
| Vague notion | Dimensional expansion |
| Specific problem | Problem inversion |
| Too many ideas | Convergence prompts |
| Feels derivative | Analogy mining, constraint removal |
| Stuck on format | Timeline projection, hybridization |
| Multiple audiences | Audience stretching |
