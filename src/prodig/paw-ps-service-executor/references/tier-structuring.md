# Tier Structuring

## Purpose

Design Good/Better/Best service tiers that serve different client segments while maximizing revenue and client fit.

## Why Three Tiers?

| Single Option | Two Options | Three Options |
|---------------|-------------|---------------|
| Take it or leave it | Binary choice | Range of fit |
| No comparison anchor | No clear "best" | Clear middle target |
| Limited upsell path | Limited upsell path | Natural upgrade path |

Three tiers work because:
- Most people choose the middle
- High tier anchors perceived value
- Low tier serves price-sensitive clients
- Clear progression between tiers

## Tier Framework

### Naming Conventions

| Approach | Tier 1 | Tier 2 | Tier 3 |
|----------|--------|--------|--------|
| Descriptive | Starter | Standard | Premium |
| Outcome | Foundation | Growth | Scale |
| Delivery | DIY | Done-With-You | Done-For-You |
| Custom | [Creative name] | [Creative name] | [Creative name] |

### Price Anchoring

```
Premium (High Anchor): $15,000
Standard (Target): $7,500 ← Most choose this
Basic (Entry): $3,000
```

Price spacing:
- Tier 2 should be roughly 2-3x Tier 1
- Tier 3 should be roughly 2-3x Tier 2
- Middle tier is where you want most sales

## Tier Design Patterns

### Pattern 1: Delivery Depth

Same outcome, different levels of service:

```
CONTENT STRATEGY PACKAGE

DIY ($1,500)
- Complete framework templates
- Video training (3 hours)
- Example strategies
- Email support

Done-With-You ($4,500)
- Everything in DIY
- 3 strategy calls (60 min each)
- Personalized feedback
- 2 revision rounds

Done-For-You ($12,000)
- Everything in DWY
- Complete strategy creation
- Implementation roadmap
- Team alignment session
- 30-day support
```

### Pattern 2: Scope Breadth

Different coverage levels:

```
MARKETING AUDIT

Starter ($2,500)
- Website review
- SEO basics
- 10-page report
- Top 5 recommendations

Standard ($6,000)
- Everything in Starter
- Social media presence
- Email marketing review
- Competitor comparison
- 30-page report
- Prioritized action plan

Premium ($15,000)
- Everything in Standard
- Paid ads audit
- Analytics deep dive
- Customer research review
- Team presentation
- 90-day implementation support
```

### Pattern 3: Outcome Scale

Different result levels:

```
FUNNEL BUILD

Launch ($3,000)
- 1 funnel (lead gen or sales)
- Up to 3 pages
- Basic email sequence (3 emails)
- Template design

Growth ($8,000)
- 2 funnels
- Up to 6 pages
- Full email sequence (7 emails)
- Custom design
- A/B test setup

Scale ($20,000)
- 3 funnels
- Up to 10 pages
- Complex sequences
- Premium design
- Full analytics
- 90-day optimization
```

### Pattern 4: Speed/Priority

Same deliverable, different timeline:

```
WEBSITE BUILD

Standard ($8,000)
- 8-week timeline
- Weekly updates
- Standard support

Accelerated ($12,000)
- 4-week timeline
- Daily updates
- Priority support

Rush ($18,000)
- 2-week timeline
- Same-day updates
- 24/7 support during build
```

## Tier Differentiation Elements

### What to Vary Between Tiers

| Element | Low Tier | Mid Tier | High Tier |
|---------|----------|----------|-----------|
| Deliverable count | Fewer | Standard | Comprehensive |
| Customization | Templates | Semi-custom | Fully custom |
| Access level | Async only | Scheduled calls | Priority access |
| Revisions | 1 round | 2 rounds | Unlimited |
| Timeline | Standard | Standard | Accelerated |
| Support duration | None | 14 days | 30-90 days |
| Team involvement | Self-service | Specialist | Senior lead |

### What to Keep Consistent

- Core methodology/approach
- Quality standards
- Brand alignment
- Professional communication
- Basic documentation

## Tier Pricing Math

### Target Distribution

```
Expected tier selection:
- Tier 1: 20% of clients
- Tier 2: 60% of clients ← Target
- Tier 3: 20% of clients

Revenue contribution:
- Tier 1: $3K × 20 = $60K (from 20 clients)
- Tier 2: $7.5K × 60 = $450K (from 60 clients)
- Tier 3: $15K × 20 = $300K (from 20 clients)
─────────────────────────────
Total: $810K from 100 clients
```

### Upsell Path

```
Initial purchase → Upsell opportunity

DIY → Add coaching calls ($500 each)
DWY → Add implementation support ($X/month)
DFY → Add ongoing retainer ($X/month)

Every tier should have upsell options.
```

## Tier Mistakes

| Mistake | Problem | Fix |
|---------|---------|-----|
| Low tier too good | No one buys mid/high | Reduce low tier value |
| High tier too expensive | Pricing seems unrealistic | Anchor with market data |
| Middle tier weak | People choose ends | Strengthen middle value |
| Too many tiers | Decision paralysis | Stick to 3-4 max |
| Unclear differences | Hard to compare | Use comparison table |
| Low tier cannibalizes | Hurts overall revenue | Make mid tier clearly better |

## Tier Review Checklist

- [ ] Three tiers clearly defined
- [ ] Price progression makes sense (2-3x steps)
- [ ] Each tier has clear "best for" client type
- [ ] Differences are obvious and valuable
- [ ] Middle tier is attractive target
- [ ] High tier anchors perceived value
- [ ] Low tier serves budget-conscious without devaluing
- [ ] Comparison table makes differences obvious
- [ ] Each tier has upsell options
